# Basic Admin Settings Application

Version: 1.0.0

This is the first round of development using a vague prompt. The application includes:
- Admin authentication (HTTP Basic Auth)
- Settings form with server-side and client-side validation
- Dashboard to display submitted settings data
- Persistent storage via data.json file
- Tailwind CSS for styling

## How to run

```bash
npm install
node server.js
```

Open http://localhost:3000 and authenticate with `admin` / `admin123`.
